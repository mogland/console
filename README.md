## Usage

```bash
pnpm i
pnpm build
```

## Development

```bash
# Mog Core
pnpm i
pnpm start:core
pnpm start:page-service
pnpm start:user-service
## Then, run user init program firstly

# Mog Console
pnpm i
pnpm dev
```
